Perguntas frequentes

Por que criar a conta antes de baixar?

Pra você usar a mesma conta no painel depois. O instalador tem ~540 MB porque já vem com 500 efeitos prontos pra usar — assim você não espera download na hora de editar. Criar conta é grátis.

"Apple não pôde verificar este arquivo" — o que faço?

É normal pra software sem certificado Apple Developer (em processo aqui).

Temos um método caso seu Mac não permita o download:

Abra o Terminal e cole:

xattr -dr com.apple.quarantine ~/Downloads/CinePRO.pkg

Depois é só duplo-clique normal no .pkg que vai funcionar.

"O Windows protegeu seu PC" (SmartScreen) — o que faço?

Mesma coisa do Mac: clique em "Mais informações" → "Executar mesmo assim". Acontece com qualquer instalador novo até o SmartScreen acumular reputação ou recebermos certificado EV.
